<h2><b>Travel Assistant Project - (Groq + Streamlit)</b></h2>

An AI-powered Travel Assistant built using Streamlit, Groq LLM, and DuckDuckGo Search.
The app helps users plan travel by answering questions and fetching real-time travel information when needed.

**Features**
1. Chat-style conversational UI (ChatGPT-like experience)
2. Uses LLM knowledge (Groq LLM) for general questions
3. Uses web search via DuckDuckGo for up-to-date travel info

**Tech Stack**
1. Python
2. Streamlit – Frontend UI
3. Groq (LLaMA 3.1)
4. LangChain
5. DuckDuckGo Search
6. python-dotenv – Environment variable management
